Official website of Barad√©'s map World of Warcraft Reforged for the game Warcraft III: Reforged. World of Warcraft Reforged is an Open World RPG multiplayer custom map inspired by the MMORPG game World of Warcraft and combines RTS and RPG game genres. It is based on the map World of Warcraft Reborn by EvilPitlord and DeMoNiKuS.

Download

  • Releases on GitHub (see "Assets" - wowr[version].w3x)
  • Downloads on Warcraft 3: Reforged - Map database
  • Downloads on Google Drive
Download the map file (wowr[version].w3x) and move it into Warcraft's folder for downloaded custom maps: 
C:\Users\%USERNAME%\Documents\Warcraft III\Maps\Download
The map is unprotected and not optimized. Join our Discord Server to find other players and discuss the map.
