Download

  • Downloads on GitHub (see "Assets" - wowr[version].w3x)
  • Download on Warcraft 3: Reforged - Map database
  • Download development and archived versions on GitHub
Download the map file (wowr[version].w3x) and move it into Warcraft's folder for downloaded custom maps: 
C:\Users\%USERNAME%\Documents\Warcraft III\Maps\Download
The map is unprotected and not optimized.
