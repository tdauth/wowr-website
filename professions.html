Professions allow you to craft items. The first profession has to be chosen in the beginning of the game.
Icon Name Description Item
Herbalist Allows crafting items which heal your hero and increase the life.
Alchemist Allows crafting items which restore mana of your hero and increase the maximum mana.
Weapon Smith Allows crafting items which increase the damage of your hero.
Armorer Allows crafting items which increase the armor of your hero.
Engineer Allows crafting tiny towers to protect your base.
Lore Master Allows crafting scrolls.
Sorcerer Allows crafting wands.
Runeforger Allows crafting runes.
Dragon Breeder Allows crafting dragon eggs.
Jewelcrafter Allows crafting items which increase your hero attributes.
Enchanter Allows crafting items which increase the bonuses of carried items. Recommended as secondary profession since it helps to improve other items. Use "-enchanter" to list all item types which can be enchanted.
Pirate Allows crafting items which allow you to live as a pirate. Arrr!
Archaeologist Allows digging for random items.
Witch Doctor Allows crafting wards.
Merchant Allows crafting bills to increase charges of other items.
